# Trackflation

Analyze historical World Athletics top times and project future world records.

## Features
- **Historical Scraper**: Collects top track times since 1974.
- **Trend Analysis**: Quantifies the rate of improvement ("inflation") per event.
- **Future Projections**: Estimates WRs for the next 20+ years using time-series models.

## Setup
1. `python -m venv venv`
2. `.\venv\Scripts\activate`
3. `pip install -r requirements.txt`
